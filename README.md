# bili-downloader

一个极简的B站视频下载器demo（带GUI）

> - 仅用于学习，功能毕竟比较敏感，所以不打算打包Exe，<del>感兴趣可以自学python</del>
> - 至于登录功能主要是太危险了所以不做，但是可以在设置里手动填写cookie

隐藏功能：命令行参数可设置 `proxy` (公司网络把B站ban了，设置代理才能用)
(目前还有BUG，建议当这个功能不存在……)

**优点**：

1. 开源，无广告
2. 轻量，没那么多乱七八糟的功能
3. 会不断维护更新，以保证可用
